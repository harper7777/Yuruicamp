# Yuruicamp｜品牌設計定調 v2.2｜莫蘭迪霧森林色調版

> 以「療癒系文青奶油風 × 日系露營選物 × 北歐自然生活感」為基礎，加入約 80% 的莫蘭迪灰霧調性，保留 Yuruicamp 所需的鬆弛、親切與新手友善；色彩分工維持清楚：**莫蘭迪綠負責品牌與信任，營火陶土色負責行動與轉換。**

---

## 1. 品牌核心定位

### 品牌一句話

> **不用準備得很完美，也可以舒服地去露營。**

Yuruicamp 不是強調征服自然、裝備競賽或極限冒險的戶外品牌，而是一個讓人用自己的步調接近自然的露營生活品牌。

品牌提供的不只是露營用品，而是：

- 更容易開始的露營方式
- 更舒服的戶外生活選擇
- 不製造裝備焦慮的購物體驗
- 有人陪著挑選、理解與使用裝備的安心感
- 讓第一次露營也能輕鬆完成的小小成就感

### 品牌定位

```text
Soft Outdoor Lifestyle
輕鬆自在的戶外生活品牌
```

整體風格定義：

> **療癒系文青奶油風 × 日系露營選物 × 北歐自然生活感 × 輕鬆新手友善**

核心不是「戶外冒險」，而是：

> **把舒服的日常帶到戶外。**

### 品牌標語

```text
Camp at Your Own Pace.
按照自己的步調去露營。
```

中文品牌主張：

```text
露營，不需要太用力。
```

可交替使用：

```text
把舒服帶到戶外。
慢慢準備，也能好好出發。
第一次露營，也可以很輕鬆。
不需要懂很多，也能開始露營。
週末，慢慢來。
```

## 2. 品牌希望帶給人的感受

Yuruicamp 的視覺與體驗應依照以下優先順序建立：

1. 放鬆
2. 溫暖
3. 親切
4. 自然
5. 簡單
6. 安心
7. 有一點可愛與幽默

使用者第一眼應感受到：

> 「這裡看起來很舒服，我不需要很專業也可以開始。」

第二眼才注意商品與功能。

### 應出現的感受

```text
慢生活、森林晨光、奶油帳篷、原木、亞麻、手沖咖啡、
週末露營、生活選物、自然材質、柔和留白、新手友善、
不用太認真、小小冒險、有人陪著準備
```

### 不應出現的感受

```text
征服自然、極限冒險、軍事戰術、裝備炫耀、高壓促銷、
黑橘硬派、冷冽科技、昂貴距離感、複雜難懂、只有老手才適合
```

## 3. 品牌人格與語氣

可以把 Yuruicamp 想像成一個人：

> 喜歡咖啡、簡單料理與週末郊外。懂得挑選好用的東西，但不賣弄專業；不追求把行程排滿，也不要求每次露營都完美。朋友第一次露營時，會幫忙準備一份剛剛好的清單，提醒對方帶外套，然後說：「沒關係，慢慢來就好。」

### 品牌人格比例

```text
35% 溫柔自然
25% 輕鬆親切
20% 日系生活感
10% 專業可靠
10% 可愛幽默
```

品牌不宜變得過度精緻、疏離或像精品選物店。

### 品牌說話方式

應該：

- 溫和、有耐心
- 不說教、不賣弄術語
- 不製造裝備焦慮
- 提供簡單、清楚的選擇
- 偶爾帶一點輕鬆幽默
- 像懂露營的朋友，而不是嚴格的教練

避免：

- 「露營必備，不買會後悔」
- 「專業玩家唯一選擇」
- 「頂級機能，征服所有地形」
- 「最後倒數，錯過不再」
- 過度使用驚嘆號、倒數計時與紅色促銷

### 文案範例

```text
第一次露營，不知道從哪裡開始？
我們先幫你準備一套簡單的。

搭帳篷已經夠忙了，桌椅就選好收一點的。

不一定要走很遠，舒服就是目的地。

一張椅子、一盞燈、一杯熱飲，
就是一個剛剛好的晚上。

裝備不用一次買齊，
先帶上真正需要的就好。
```

## 5. 視覺風格定位

### 主風格

```text
Japanese Soft Outdoor Minimalism
日系柔和戶外極簡
```

搭配：

```text
Morandi Creamy Editorial
莫蘭迪奶油系生活編輯感
```

整體應像：

> 森林旁的一間小型露營生活選物店，有晨光、咖啡香、亞麻布與一點不刻意的手繪痕跡。

### 視覺比例

```text
55% 自然戶外
25% 日系生活選物
10% 北歐簡約
10% 輕鬆手繪與品牌個性
```

### Yuruicamp 專屬的「鬆弛感」

在原本精緻、安靜的視覺基礎上，加入：

- 輕微不對稱的插畫構圖
- 帶手繪感的帳篷、杯子、雲、營火與樹木線條
- 小幅度歪斜或錯位的裝飾
- 像手寫備註的短句
- 柔和而非完美幾何的圖形
- 「慢慢來」「今天也不用趕」等小文案
- 有一點慵懶表情、但不幼稚的角色或圖示

> 鬆弛感不是凌亂，而是在清楚秩序中保留一點不刻意。

### 莫蘭迪調性控制

Yuruicamp 適合帶有莫蘭迪色調，但不應將所有元素完全灰化。

建議控制在：

```text
80% 低彩度、帶灰感的背景與品牌色
20% 保留清楚對比的文字、CTA 與互動狀態
```

莫蘭迪感主要用在：

- Header、Footer 與品牌綠
- 奶油白、燕麥米與亞麻邊框
- 卡片、標籤與插畫
- 圖片的森林、木材、布料與陰影
- 次要按鈕與非交易型互動

不應過度灰化：

- 加入購物車、預訂與結帳 CTA
- 主要標題與重要資訊
- 表單 Focus、錯誤與選中狀態
- 庫存、日期、價格與付款資訊

> 莫蘭迪是降低躁動感，不是降低辨識度。

整體色彩主題定義為：

> **莫蘭迪霧森林 × 奶油亞麻 × 營火陶土**

## 6. 目標色標系統

### 色彩核心概念

> **莫蘭迪奶油白建立空間，亞麻灰米增加溫度，霧森林綠建立品牌辨識，深灰綠維持可讀性，營火陶土色負責重要行動。**

不再使用「綠色、黃色、咖啡色」三個彼此競爭的主題色，而是讓色彩明確分工：

```text
莫蘭迪綠：品牌識別、信任、瀏覽與一般互動
陶土色：加入購物車、預訂、結帳、送出等轉換行動
奶油與亞麻色：承載內容與建立空間
```

陶土色不是另一個大面積主題色，而是全站固定、少量且高辨識的轉換色。

### 色彩使用比例

```text
60% 莫蘭迪奶油白與暖霧白
23% 奶茶、亞麻與燕麥灰米
10% 莫蘭迪森林綠
 4% 深灰綠文字與功能色
 3% 營火陶土 CTA 與狀態色
```

陶土色雖只占約 3%，但因色相與鼠尾草綠有明顯差異，已足以形成視覺焦點。CTA 的效果來自「少量、固定、只出現在重要行動」，而不是大面積使用。

### 陶土色角色分工

陶土色分為兩種用途，不能混用：

#### 轉換 CTA 色

```text
#A86549  營火陶土色
#8E543D  CTA Hover
#7D4935  CTA Active
#EFE0D6  CTA Soft
```

用於：

- 開始挑選裝備
- 加入購物車
- 立即預訂
- 確認預訂
- 前往結帳
- 完成付款
- 送出租借申請
- 訂閱電子報

#### 插畫與人情溫度色

```text
#AD806C  柔和陶土
```

只用於：

- 營火、杯子等手繪插畫
- 小型品牌裝飾
- 重要但非危險的提示圖示
- 不超過整頁約 1%–2% 的細節

禁止：

- 陶土色大面積鋪底
- Header 或 Footer 全面改成橘色
- 所有按鈕都使用陶土色
- 標籤、圖示與裝飾大量搶用 CTA 色
- 使用高飽和亮橘、螢光橘或偏黃色的橘色

## 7. UI 元件設計定調

### 7.1 第一層：Conversion CTA

只用於使用者即將採取的重要轉換行動：

- 開始挑選裝備
- 加入購物車
- 立即預訂
- 確認預訂
- 前往結帳
- 完成付款
- 送出申請
- 訂閱電子報

```css
.btn-cta {
  background: #A86549;
  color: #FCFAF5;
  border: 1px solid #A86549;
}
```

Hover：

```css
.btn-cta:hover {
  background: #8E543D;
  border-color: #8E543D;
  transform: translateY(-1px);
}
```

Active：

```css
.btn-cta:active {
  background: #7D4935;
  border-color: #7D4935;
  transform: translateY(0);
}
```

Focus：

```css
.btn-cta:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(168, 93, 58, 0.22);
}
```

### 7.2 第二層：品牌與瀏覽型按鈕

用於不會立即產生交易的操作：

- 查看商品
- 查看推薦組合
- 閱讀新手指南
- 探索露營日誌
- 修改內容
- 查看詳細資訊

實心版本：

```css
.btn-brand {
  background: #647260;
  color: #FCFAF5;
  border: 1px solid #647260;
}
```

Hover：

```css
.btn-brand:hover {
  background: #566452;
  border-color: #566452;
}
```

### 7.3 第三層：Secondary Button

適合「先看看」「返回」「修改」「閱讀更多」：

```css
.btn-secondary {
  background: #FCFAF5;
  color: #647260;
  border: 1px solid #899486;
}
```

Hover：

```css
.btn-secondary:hover {
  background: #DDE3D8;
}
```

### 7.4 第四層：Tertiary Action

用於取消、稍後決定、查看更多與文字型操作：

```css
.btn-tertiary {
  background: transparent;
  color: #5F5A53;
  border: none;
  text-decoration: underline;
  text-underline-offset: 4px;
}
```

### 7.5 按鈕文字與尺寸

```text
高度：44px–48px
圓角：14px 或膠囊形
字重：500–600
主要 CTA 可比次要按鈕略寬或略高
同一區塊最多只能有一個陶土色主要 CTA
```

推薦文字：

```text
第一次露營從這裡開始
開始挑選裝備
加入購物車
立即預訂
確認預訂
前往結帳
看看推薦組合
閱讀新手指南
```

避免：

```text
立即搶購
錯過不再
終極方案
最後倒數
```

### 7.6 商品卡 CTA 規則

首頁為了維持安靜感，商品卡平常可使用綠色外框；Hover 或使用者聚焦時，再轉為陶土色：

```css
.product-card .add-cart {
  background: #FCFAF5;
  color: #647260;
  border: 1px solid #899486;
}

.product-card:hover .add-cart,
.product-card .add-cart:focus-visible {
  background: #A86549;
  color: #FCFAF5;
  border-color: #A86549;
}
```

正式商品列表、商品詳情、購物車與結帳頁，可直接使用陶土色「加入購物車／預訂／結帳」按鈕。

### 7.7 CTA 不只靠顏色

- 一個區塊最多只有一個主要 CTA
- 主要按鈕周圍應有足夠留白
- 主要按鈕尺寸略高於次要按鈕
- 文案需明確說明點擊後會發生什麼
- 不讓標籤、插畫與一般圖示搶用 CTA 色
- Hover、Focus、Active 狀態必須清楚
- 陶土色只出現在真正重要的下一步

### 卡片

```css
background: #FCFAF5;
border: 1px solid rgba(220, 207, 192, 0.72);
border-radius: 18px;
box-shadow: 0 3px 14px rgba(62, 70, 61, 0.045);
```

Hover：

```css
box-shadow: 0 8px 24px rgba(62, 70, 61, 0.10);
transform: translateY(-2px);
```

### 標籤

```text
新手友善：#DDE3D8 / #3F473E
輕鬆好收：#E6DED2 / #5F5A53
本週選物：#AEB9A9 / #3F473E
一個人也適合：#EDE7DD / #5F5A53
```

避免紅色、亮黃色與大量「熱賣」「搶購」角標。

### 表單

```css
background: #FCFAF5;
border: 1px solid #D4C9BB;
color: #3F473E;
border-radius: 12px;
```

Focus：

```css
border-color: #899486;
box-shadow: 0 0 0 3px rgba(159, 170, 150, 0.18);
```

錯誤訊息應使用溫和語氣：

```text
這裡好像少填了一點資料。
再確認一次日期就可以囉。
```

## 8. 版面設計定調

### Header

Yuruicamp 可以使用綠色 Header，並建議採用帶灰、偏暖的莫蘭迪森林綠。

#### 最推薦的 Header 色

```text
#647260  莫蘭迪森林綠
```

它比鮮綠安靜、比軍綠柔和，也比淺鼠尾草綠更能承載奶油白小字。

```css
.site-header {
  background: rgba(100, 114, 96, 0.96);
  color: #FCFAF5;
  border-bottom: 1px solid rgba(252, 250, 245, 0.14);
  backdrop-filter: blur(10px);
}
```

Logo：

```css
.site-logo {
  color: #FCFAF5;
}
```

導覽文字：

```css
.site-nav a {
  color: rgba(252, 250, 245, 0.76);
}

.site-nav a:hover,
.site-nav a.active {
  color: #FCFAF5;
}
```

Active 可使用奶油白細線或淡綠膠囊，不使用亮黃、金棕或高飽和色塊。

Header 中若有一個重要轉換行動，例如「開始預訂」，可使用營火陶土色：

```css
.header-cta {
  background: #A86549;
  color: #FCFAF5;
  border: 1px solid #A86549;
}

.header-cta:hover {
  background: #8E543D;
  border-color: #8E543D;
}
```

同一個 Header 最多只有一個陶土色 CTA。登入、收藏、搜尋與購物車圖示應維持奶油白或淺亞麻色。

### Hero

Hero 以生活場景照片為主，不使用大面積深綠色主視覺。

照片內容：

- 奶油白帳篷
- 柔和晨光
- 低對比森林
- 原木與亞麻
- 咖啡、書、毯子、燈具
- 人物正在準備、聊天、閱讀或休息
- 不需要過度完美的擺拍

遮罩：

```css
background: linear-gradient(
  90deg,
  rgba(250, 249, 238, 0.84) 0%,
  rgba(250, 249, 238, 0.60) 38%,
  rgba(250, 249, 238, 0.10) 72%
);
```

Hero 文案：

```text
把日子，慢慢搬進森林。

不需要準備得很完美，
```

### Logo 方向

或：

推薦：

## 11. 字體、圓角、陰影與圖片

### 字體

```css
--font-heading: "Noto Serif TC", serif;
--font-body: "Noto Sans TC", sans-serif;
```

字重：

```text
主標題：600
區塊標題：600
內文：400
按鈕：500–600
```

手寫感字體只可少量用於備註、插畫旁短句或社群素材。

### 圓角

```text
小型元件：10px
輸入框：12px
按鈕：14px 或膠囊形
商品卡片：18px
大型區塊：24px
```

### 陰影

```css
--shadow-soft: 0 3px 14px rgba(62, 70, 61, 0.045);
--shadow-hover: 0 8px 24px rgba(62, 70, 61, 0.10);
```

### 圖片風格

符合：

```text
柔和晨光、低飽和、奶油霧感、自然陰影、原木、亞麻、
陶瓷、手沖咖啡、書籍、毯子、森林與草地、真實使用狀態
```

人物狀態：

- 坐著休息
- 慢慢準備餐點
- 閱讀
- 喝咖啡
- 和朋友聊天
- 不需要看鏡頭

避免：

```text
高反差 HDR、鮮豔橘色帳篷、軍綠迷彩、黑色戰術裝備、
極限登山、冷色棚拍、完美到不真實的擺拍
```

莫蘭迪圖片調色原則：

- 整體降低彩度，但人物膚色不可變灰
- 森林綠壓低黃感與螢光感
- 木材改為灰棕，不偏紅橘
- 奶油帳篷與亞麻保持乾淨，不泛黃
- 陰影不使用死黑，保留柔和層次
- CTA 商品與重要資訊仍需保持清楚辨識

## 12. 電商操作原則

Yuruicamp 可以柔和，但不能模糊。

- 主要轉換 CTA 使用 `#A86549`
- CTA Hover 使用 `#8E543D`
- CTA Active 使用 `#7D4935`
- 品牌與瀏覽型按鈕使用 `#647260`
- 綠色 Hover 使用 `#566452`
- 選中狀態使用 `#DDE3D8`
- Focus 狀態必須清楚
- 價格、庫存、日期與規格不可因風格而降低可讀性
- 加入購物車、預訂與結帳不可和一般「查看詳情」使用相同層級

商品卡層級：

```text
情境標籤
商品圖片
商品名稱
適合對象／使用難度
價格
主要行動
收藏
```

促銷文案推薦：

```text
本週慢慢選
週末剛好適合
新手組合少一點準備
一起帶走比較省事
```

避免：

```text
最後一小時
瘋狂下殺
買到賺到
倒數搶購
紅色滿版折扣
```

## 15. 禁止事項

### 色彩

禁止：

- 高飽和黃色
- 高飽和亮橘、螢光橘或偏黃橘色 CTA
- 大面積深咖啡色
- 黑橘戶外配色
- 螢光綠
- 冷科技藍
- 所有顏色都過度灰化，導致畫面疲倦與 CTA 消失
- 深咖啡或黃銅色 icon 直接置於深綠 Header／Footer
- 三種主題色平均分配
- 每個圖示使用不同顏色

### 版面

禁止：

- 深綠色滿版 Hero
- 過多促銷模組
- 元件擁擠
- 大量粗框線
- 高對比陰影
- 所有區塊都使用卡片包住
- 只有漂亮照片，卻沒有新手引導

### 文案

禁止：

- 製造焦慮
- 賣弄術語
- 對新手說教
- 過度高級、疏離
- 太多英文戶外術語
- 把露營說成需要完整裝備才能開始

## 17. 最終設計原則

> **以莫蘭迪奶油白建立空間，以亞麻灰米增加溫度，以霧森林綠建立品牌識別，以墨灰綠維持文字可讀性，以營火陶土色指出重要的下一步。**

> **在安靜精緻的視覺中，加入一點鬆、一點親切，以及第一次露營也能安心開始的陪伴感。**

> **莫蘭迪森林綠讓使用者認得品牌，營火陶土色讓使用者知道下一步。**

最終想呈現的不是：

> 在森林裡，完美地過一段精緻生活。

而是：

> **不需要準備得很完美，也可以用自己的步調，舒服地去露營。**
